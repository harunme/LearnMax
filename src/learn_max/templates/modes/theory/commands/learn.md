使用 LearnMax 框架为指定主题初始化以理论为重点的学习。

- 提示："此项目已在学习 [主题名称]"
- 首先检查待复习内容（如有待复习，在新学习前进行）
- 继续当前主题（1 个项目 = 1 个学习目标）

**如果主题尚不存在：**

2. 运行：`python3 .learning/scripts/init_learning.py "[主题名称]" .learning`
3. 解析 JSON 输出并遵循 `llm_directive`
4. **阅读** `.learning/<主题-slug>/syllabus.md` 查看模板结构
5. 根据用户的深度和风格生成**以概念为中心的大纲**
6. **替换** 模板占位符为实际内容
7. 更新元数据：在 `.learning/<主题-slug>/metadata.json` 中设置 `"syllabus_generated": true`

**理论导向大纲结构：**

- **概念概述：** 全局图景、为什么这个主题重要、它回答的关键问题
- **先决条件：** 你应该先理解的概念（附解释）
- **核心原则：** 一切构建所在的 3-5 个基本真理
- **学习旅程：** 按概念构建块组织
  - 每个阶段：概念 → 直觉 → 第一性原理 → 心智模型 → 联系
  - 包含 🤔 思想实验和 📊 可视化练习
  - 包含 ✅ 复选框用于"我能简单解释这个"
- **概念图：** 所有想法如何联系（文本中的可视化表示）
- **深度探讨主题：** 为好奇学习者可选的高级领域
- **常见误解：** 人们会犯什么错误及为什么
- **成功标准：** "我能从第一性原理解释 X"而非"我记住了 X"

**重要：**

- 生成全面的概念图（不仅仅是列表）
- 每个概念都包含"为什么"（不仅仅是"是什么"）
- 建立逻辑进展（每个概念建立在前一个之上）
- 添加测试理解的思想实验
- 包含边缘情况和边界条件
- 跨领域联系相关概念

## 大纲生成后

1. **激活先验知识：**
   - "你已经对 [主题] 了解多少？"
   - "你当前的心智模型是什么？"
   - 从他们现有的理解开始
   - 温和地识别误解

2. **建立基础：**
   - 从"[主题] 为什么存在？"开始
   - 它解决什么问题？
   - 没有它世界会是什么样子？
   - 在机制之前创造动机

3. **第一个概念会话：**
   - 选择最基础的概念
   - 通过类比建立直觉（15 分钟）
   - 从第一性原理探索（20 分钟）
   - 用思想实验测试理解（10 分钟）
   - 让用户讲解回去（15 分钟）
   - 一起创建可视化表示

## 持续学习模式

**每次学习会话（60 分钟）：**

1. **激活**（5 分钟）：
   - 回顾之前的概念
   - "你会如何解释 [昨天学的概念]？"

2. **建立直觉**（15 分钟）：
   - 新概念介绍
   - 真实世界类比
   - "这让你想起了什么？"

3. **第一性原理**（20 分钟）：
   - 分解到基本要素
   - "为什么这必须是真的？"
   - 逻辑上构建概念

4. **思想实验**（10 分钟）：
   - "如果我们改变 [参数] 会怎样？"
   - 测试边界和边缘情况
   - 找到模型在哪里崩溃

5. **讲解**（10 分钟）：
   - 用户用自己的话解释
   - 不用术语
   - 使用类比或图表

**每周模式：**

- 第 1-3 天：新概念深度探索
- 第 4 天：联系概念，建立更大图景
- 第 5 天：从不同角度重访概念
- 第 6 天：向某人讲解概念（或写解释）
- 第 7 天：探索高级/边缘情况或休息

**每月复习：**

- 从第一性原理解释整个主题
- 建立完整概念图
- 识别剩余问题
- 探索一个深度主题

## 深度探讨会话

定期提供：

- "想探索 [高级主题] 吗？"
- "让我们检验一个边缘情况"
- "如果质疑这个假设会怎样？"
- "历史背景：这是如何被发现的？"

## 成功指标

当用户做到这些时就是成功的：

- 不用术语简单解释概念
- 自发地使用准确的类比
- 独立地问"为什么"问题
- 将新概念与现有知识联系起来
- 正确预测新场景中的行为
- 识别假设和局限性
- 表现出超越大纲的真正好奇心

## 教学时刻

定期提示：

- "向非技术朋友解释这个"
- "一句话的核心思想是什么？"
- "画一张描述这个如何运作的图片"
- "什么真实世界的例子说明了这个？"
- "如果你在教这个，你会强调什么？"

**记住：** 这是理论模式。深度理解优于广度。质量优于数量。一个真正掌握的概念可以简单解释、灵活应用、广泛联系。鼓励好奇心。即使"超出大纲"也要跟随有趣的线索。
