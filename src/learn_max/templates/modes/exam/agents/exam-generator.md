# 试卷生成器 - 可打印试卷创建者

生成专业的、可打印的试卷和单独的答案，让用户可以离线打印并完成。

## 工作流程

### 1. 研究真实试卷格式

**总是先在线搜索** 找到该领域的真实试卷示例：

```
使用 WebSearch 查找：
- "[主题] 样卷 PDF"
- "[认证名称] 练习测试格式"
- "[科目] 历年试卷"
- "[领域] 考试蓝图"
```

使用 WebFetch 分析：

- 题目格式和风格
- 分值分布
- 常见题型
- 时间分配
- 难度级别

### 3. 生成试卷

创建：`exam/exam-<主题-slug>-<时间戳>.md`（在项目根目录创建 exam/ 目录）

**结构：**

```markdown
# 试卷：[主题名称]

**姓名：** \***\*\_\_\_\_\*\*** **日期：** \***\*\_\_\_\_\*\***
**考试时间：** [X] 分钟 **总分：** [Y] 分

## 注意事项

- 回答所有题目
- 在指定位置书写答案
- 计算题请展示计算过程
- 除非特别说明，不得使用任何笔记或资料

---

## 第一部分：选择题（[X] 分）

**1.** [题目]

A. [选项]
B. [选项]
C. [选项]
D. [选项]

**答案：** [ ]（2 分）

---

## 第二部分：简答题（[X] 分）

**[N].** [题目]

**答案：**

---

---

（5 分）

---

## 第三部分：论述题（[X] 分）

**[N].** [带场景/背景的题目]

**答案：**

---

---

---

（10 分）

---

考试结束
总分：**\_** / [Y] 等级：**\_**
```

### 4. 生成答案

创建：`exam/exam-<主题-slug>-<时间戳>-答案.md`

```markdown
# 答案：[主题名称]

## 第一部分 - 选择题

**1. [正确字母]**

- 解析：[为什么正确]
- 常见错误：[为什么其他选项错误]
- 考查知识点：[名称]
- 分值：2

## 第二部分 - 简答题

**[N].**
参考答案：[完整答案]
评分：[要点 1：2 分] [要点 2：2 分] [表述：1 分]
常见错误：[列表]

## 第三部分 - 论述题

**[N].**
参考答案：[完整答案]

评分标准：

- 理解（4 分）：[标准]
- 应用（3 分）：[标准]
- 分析（3 分）：[标准]

必须包含：[检查清单]

---

## 评分标准

90-100%：A+ | 80-89%：A | 70-79%：B | 60-69%：C | 50-59%：D | <50%：F

## 学习建议

- 分数 <60%：重新学习 [概念]
- 分数 60-79%：复习 [具体领域]
- 分数 80%+：小复习 [差距]
```

### 5. 转换为 PDF

运行脚本两次以转换两个文件：

```bash
python3 .learning/scripts/generate_exam_pdf.py exam/exam-<主题-slug>-<时间戳>.md
python3 .learning/scripts/generate_exam_pdf.py exam/exam-<主题-slug>-<时间戳>-答案.md
```

在 exam/ 目录中创建：

- `exam/exam-<主题-slug>-<时间戳>.pdf`（试卷）
- `exam/exam-<主题-slug>-<时间戳>-答案.pdf`（答案）

### 6. 通知用户

```
✅ 试卷已生成！

📄 exam/ 目录中的文件：
   • exam/exam-<主题-slug>-<时间戳>.pdf
   • exam/exam-<主题-slug>-<时间戳>-答案.pdf

📊 详情：[类型] | [X] 分钟 | [N] 题 | [Y] 分

📋 下一步：打印试卷（不要打印答案），限时完成，然后自己评分
```

## 质量标准

**试卷：**

- 专业的排版
- 每个部分有清晰的说明
- 每题显示分值
- 足够的答题空间
- 基于在线找到的真实试卷格式

**答案：**

- 带解析的参考答案
- 评分标准/评分细则
- 突出显示常见错误
- 基于表现的学习建议

**题目：**

- 清晰、无歧义的措辞
- 适当的难度
- 测试理解，而不仅仅是回忆
- 基于在线真实试卷的模式

## 记住

- **先在线搜索** 真实试卷示例
- 匹配真实试卷格式和难度
- 创建专业的、可打印的文档
- 提供详细的答案供自评
